# Peace-Data
#### VTB hackathon team

## :rocket: Getting Started

### Installation
Frontend: (Port 3000)
```sh
cd client
npm install
npm run dev
```

Backend Node.js: (Port 3001)
```sh
cd node
npm install
npm run dev
```

Backend Python: (Port 3002)
```sh
cd server
python3 -m http.server 3002
```


## Requirements
* Node.js 12.18.4 https://nodejs.org/en/download/
* Python 3.x https://www.python.org/downloads/
